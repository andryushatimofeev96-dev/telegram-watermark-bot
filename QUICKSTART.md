# Быстрый старт

## 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

## 2. Настройка config.py

Откройте `config.py` и укажите токен бота:

```python
# Токен бота от @BotFather
BOT_TOKEN = "ваш_токен"

# ID администратора (уже указан: 8320384091)
ADMIN_ID = 8320384091
```

**Всё!** База данных SQLite создастся автоматически при первом запуске.

## 3. Запуск бота

```bash
python bot.py
```

Вы должны увидеть:
```
Successfully connected to SQLite database: bot_database.db
Database tables created successfully
Бот запущен!
```

## 4. Первое использование

### Для администратора (ID: 8320384091):

1. Откройте бота в Telegram
2. Отправьте `/start`
3. Откройте админ-панель: `/admin`
4. Добавьте пользователей по их Telegram ID

### Для обычных пользователей:

1. Администратор должен добавить ваш ID через `/admin`
2. Отправьте боту `/start`
3. Добавьте водяной знак: `/add_watermark`
4. Примените его на фото: `/apply`

## Готово! 🎉

Бот готов к работе. База данных SQLite работает автоматически - никаких дополнительных настроек не требуется!

## Полезные команды

- `/admin` - Админ-панель (только для администратора)
- `/add_watermark` - Добавить водяной знак
- `/apply` - Наложить водяной знак на фото
- `/list_watermarks` - Список ваших водяных знаков
- `/help` - Справка

## Преимущества SQLite

- ✅ Не требует установки сервера
- ✅ Работает "из коробки"
- ✅ Автоматическое создание базы данных
- ✅ Один файл - вся база данных
- ✅ Легкое резервное копирование
