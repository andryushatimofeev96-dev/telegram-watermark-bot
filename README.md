# Telegram Watermark Bot 🖼️

Telegram бот для наложения водяных знаков на фотографии с автоматическим масштабированием и системой контроля доступа.

## Возможности

- ✅ Сохранение нескольких ватермарков с пользовательскими названиями
- ✅ Автоматическое масштабирование ватермарка под размер фото
- ✅ Адаптивное наложение: растягивание для больших фото, пропорциональное для маленьких
- ✅ Поддержка прозрачности (PNG)
- ✅ Высокое качество обработки изображений (PNG без потерь)
- ✅ Отправка результата как документ для сохранения качества
- 🔐 **Система контроля доступа с SQLite**
- 👑 **Админ-панель для управления пользователями**
- 🚀 **Работает "из коробки" - не требует установки сервера БД**

## Установка

1. Установите Python 3.8 или выше

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Создайте бота в Telegram:
   - Откройте [@BotFather](https://t.me/BotFather)
   - Отправьте команду `/newbot`
   - Следуйте инструкциям и получите токен

4. Настройте бота:
   - Откройте файл `config.py`
   - Замените токен на ваш

## Запуск

```bash
python bot.py
```

База данных SQLite создается автоматически при первом запуске!

## Использование

### Для обычных пользователей

#### 1. Добавление ватермарка

```
/add_watermark
```
- Отправьте фото ватермарка (как файл для сохранения прозрачности)
- Введите название (например: `slot1`, `logo`, `signature`)

#### 2. Наложение ватермарка

```
/apply
```
- Выберите ватермарк из списка
- Отправьте фото (как обычное фото или как файл)
- Получите результат как PNG-файл!

#### 3. Управление ватермарками

```
/list_watermarks - Показать все сохраненные ватермарки
/delete_watermark - Удалить ватермарк
/help - Справка
```

### Для администратора

Администратор (ID: 8320384091) имеет доступ к админ-панели:

```
/admin - Открыть админ-панель
```

**Возможности админ-панели:**
- ➕ Добавить пользователя по Telegram ID
- ➖ Удалить пользователя (забрать доступ)
- 📋 Просмотреть список всех пользователей с доступом

**Важно:** Только пользователи, добавленные администратором, могут использовать бота!

## Система доступа

### Как это работает:

1. **Администратор** (ID: 8320384091) имеет полный доступ к боту и админ-панели
2. **Обычные пользователи** должны быть добавлены администратором через `/admin`
3. Пользователи без доступа получат сообщение об отказе
4. Администратор может в любой момент забрать доступ у пользователя
5. Все данные хранятся в локальной SQLite базе данных

### Добавление пользователя:

1. Администратор открывает `/admin`
2. Нажимает "➕ Добавить пользователя"
3. Вводит Telegram ID пользователя (например: 123456789)
4. Пользователь получает доступ и может использовать бота

## Устранение неполадок

### Ошибка "Invalid token"
- Проверьте правильность токена в `config.py`
- Убедитесь, что токен получен от @BotFather

### Ошибка "У вас нет доступа"
- Обратитесь к администратору бота
- Администратор должен добавить ваш Telegram ID через `/admin`

### База данных не создается
- Проверьте права на запись в папку проекта
- Убедитесь что Python установлен правильно

### Бот не отвечает
- Проверьте интернет-соединение
- Убедитесь, что бот запущен (`python bot.py`)
- Проверьте логи на наличие ошибок
