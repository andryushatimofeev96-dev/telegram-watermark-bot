# Настройка базы данных SQLite

## Автоматическая настройка ✅

**Ничего делать не нужно!** База данных SQLite создается автоматически при первом запуске бота.

## Как это работает

1. При первом запуске бота создается файл `bot_database.db`
2. Автоматически создаются все необходимые таблицы
3. Администратор (ID: 8320384091) автоматически добавляется в список разрешенных пользователей

## Преимущества SQLite

- ✅ Не требует установки сервера
- ✅ Не требует настройки
- ✅ Работает "из коробки"
- ✅ Легкий и быстрый
- ✅ Один файл - вся база данных
- ✅ Идеально подходит для небольших ботов

## Структура базы данных

### Таблица: allowed_users

| Поле | Тип | Описание |
|------|-----|----------|
| id | INTEGER | Автоинкремент ID |
| user_id | INTEGER | Telegram ID пользователя (уникальный) |
| username | TEXT | Username пользователя |
| first_name | TEXT | Имя пользователя |
| added_at | TIMESTAMP | Дата и время добавления |

## Расположение базы данных

По умолчанию база данных находится в файле `bot_database.db` в корневой папке проекта.

Вы можете изменить путь в `config.py`:

```python
DB_PATH = "bot_database.db"  # Или любой другой путь
```

## Резервное копирование

Для создания резервной копии просто скопируйте файл базы данных:

```bash
# Windows
copy bot_database.db backup_bot_database.db

# Linux/Mac
cp bot_database.db backup_bot_database.db
```

Для восстановления замените текущий файл резервной копией.

## Просмотр базы данных

Вы можете просмотреть содержимое базы данных с помощью:

### DB Browser for SQLite (GUI)
Скачайте с https://sqlitebrowser.org/

### Командная строка
```bash
sqlite3 bot_database.db
```

Затем выполните SQL запросы:
```sql
-- Посмотреть всех пользователей
SELECT * FROM allowed_users;

-- Добавить пользователя вручную
INSERT INTO allowed_users (user_id, username, first_name) VALUES (123456789, 'username', 'Name');

-- Удалить пользователя
DELETE FROM allowed_users WHERE user_id = 123456789;
```

## Миграция с MySQL

Если у вас была MySQL база данных, вам нужно:

1. Экспортировать данные из MySQL
2. Импортировать их в SQLite вручную через SQL запросы

Или просто начать с чистой базы - администратор добавится автоматически.

## Возможные проблемы

### Ошибка "database is locked"
- Закройте все программы, которые могут использовать базу данных
- Перезапустите бота

### Файл базы данных не создается
- Проверьте права на запись в папку проекта
- Убедитесь что путь в `config.py` правильный

### База данных повреждена
- Удалите файл `bot_database.db`
- Перезапустите бота - база создастся заново
- Администратор будет добавлен автоматически
