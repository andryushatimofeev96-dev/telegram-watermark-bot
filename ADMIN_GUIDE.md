# Руководство администратора

## Ваш ID администратора: 8320384091

## Команды администратора

### `/admin` - Открыть админ-панель

После открытия панели доступны кнопки:

- **➕ Добавить пользователя** - Добавить нового пользователя
- **➖ Удалить пользователя** - Забрать доступ у пользователя
- **📋 Список пользователей** - Посмотреть всех пользователей с доступом
- **❌ Закрыть** - Закрыть админ-панель

## Как добавить пользователя

1. Откройте `/admin`
2. Нажмите "➕ Добавить пользователя"
3. Введите Telegram ID пользователя (например: `123456789`)
4. Пользователь получит доступ к боту

### Как узнать Telegram ID пользователя?

Попросите пользователя:
1. Открыть бота [@userinfobot](https://t.me/userinfobot)
2. Отправить любое сообщение
3. Бот покажет его ID

Или используйте [@getmyid_bot](https://t.me/getmyid_bot)

## Как удалить пользователя

1. Откройте `/admin`
2. Нажмите "➖ Удалить пользователя"
3. Выберите пользователя из списка
4. Подтвердите удаление

После удаления пользователь больше не сможет использовать бота.

## Просмотр списка пользователей

1. Откройте `/admin`
2. Нажмите "📋 Список пользователей"

Вы увидите:
- Имя пользователя
- Username (если есть)
- Telegram ID
- Дату добавления
- Отметку 👑 для администратора

## Важные замечания

- ⚠️ Вы не можете удалить себя из списка администраторов
- ✅ Вы автоматически добавлены в список при первом запуске
- 🔒 Только пользователи из списка могут использовать бота
- 📊 Все изменения сохраняются в SQLite базе данных

## Структура базы данных

База данных: `bot_database.db` (SQLite)
Таблица: `allowed_users`

Поля:
- `id` - Автоинкремент ID
- `user_id` - Telegram ID пользователя
- `username` - Username пользователя (если есть)
- `first_name` - Имя пользователя
- `added_at` - Дата и время добавления

## Резервное копирование

Для создания резервной копии базы данных:

```bash
# Windows
copy bot_database.db backup.db

# Linux/Mac
cp bot_database.db backup.db
```

Для восстановления замените текущий файл резервной копией.

## Изменение ID администратора

Если нужно изменить ID администратора, отредактируйте `config.py`:

```python
ADMIN_ID = 8320384091  # Замените на нужный ID
```

После изменения перезапустите бота.

## Поддержка

Если возникли проблемы:
1. Проверьте логи бота в консоли
2. Убедитесь что файл `bot_database.db` существует
3. Проверьте настройки в `config.py`
4. См. [SETUP_DATABASE.md](SETUP_DATABASE.md) для информации о базе данных
